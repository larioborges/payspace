Objectives

Create a responsive webpage according to the following specifications:

  1. Design a simple webpage layout that includes a header, a main content area, and a footer.
  2. Apply styles using CSS to make the layout visually appealing.
  3. Use SASS for compiling CSS and Bootstrap/TailwindCSS for utilities.
  4. Ensure the layout is responsive and adapts well to different screen sizes (desktop, tablet, mobile).
  5. Use flexbox for layout positioning.
  6. Include a navigation menu within the header.

Notes

I decided to implement this using plain HTML and CSS (no JS, NPM or SCSS). There is no hamburger menu in mobile since that would require JS.
