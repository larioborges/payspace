Objectives

You are provided with an array of objects representing books in a library. Implement the following tasks:

  1. Display the list of books on the webpage dynamically using JavaScript (native).
  2. Add functionality to allow users to sort the books by title or author in ascending or descending order.
  3. Implement a search/filter feature that filters books by title or author as the user types in a search input.
